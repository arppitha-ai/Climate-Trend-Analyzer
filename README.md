# 🌍 Climate Trend Analyzer

## 📊 Project Overview
This project analyzes climate data to understand temperature, rainfall, and weather patterns over time using Python and data visualization.

---

## 🧠 Objective
To study climate trends and identify changes in weather patterns using data analysis techniques.

---

## 🛠 Skills Used
Python, Pandas, NumPy, Matplotlib, Data Analysis, Jupyter Notebook

---

## 📈 Key Features
✔ Climate data analysis  
✔ Temperature and rainfall trend visualization  
✔ Pattern detection in weather data  

---

## 💡 Use of Project
This project helps in understanding climate changes and supports environmental analysis and research studies.
